# Changelog

## [1.0.3] - 2026-02-25

### Changed
- Updated `telegrambots` library to version 9.4.0 (matches Telegram API 9.4).
- Switched from `telegrambots-client` to `telegrambots-meta` dependency. Previously, only `telegrambots-meta` classes were used, making `client` files redundant.


## [1.0.4] - 2026-02-28
- Highlighting RegExp syntax in the corresponding annotation fields in IntelliJ IDEA-based IDE's.
