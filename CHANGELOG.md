# Список изменений

## [1.0.3] - 2026-02-25

### Изменено
- Обновление библиотеки `telegrambots` до версии 9.4.0 (что соответствует Telegram API 9.4).
- Переход от зависимости `telegrambots-client` к `telegrambots-meta`. Ранее в проекте использовались только классы из `telegrambots-meta`, поэтому файлы `client` были избыточны.


## [1.0.4] - 2026-02-28

### Добавлено
- Подсвечивание синтаксиса RegExp в соответствующих полях аннотаций в IDE, базирующихся на IntelliJ IDEA.
